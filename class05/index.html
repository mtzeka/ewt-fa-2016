Github .gitignore

You may have noticed the .sass-cache/ hidden directory that gets generated when running sass.

Creating a .gitignore file at the root of your repository will ignore any files/directories you list out. You can view the one I use for this repository here.

.DS_STORE
*.map
.sass-cache*/

Using back references in Mixins

@mixin colors() {
  color: blue;

  &:hover {
    color: red;
  }
}

A common usage of a mixin is for clearfix

@mixin clearfix() {
  &:after {
    content: '';
    display: table;
    clear: both;
  }
}

.container {
  @include clearfix();

  .column {
    float: left;
    width: 50%;
  }
}
