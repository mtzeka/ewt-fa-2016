Github pages

The neat thing about github repositories is you can turn them into hosted sites. To turn your repository into a github pages repository follow these instructions:

  • Navigate to your repository landing page.
  • Click on the "Settings" tab.
  • Scroll down to the "GitHub Pages" section.
  • Under the "Source" header it should say "GitHub Pages is currently disabled."
  • Select the "master branch" option from the dropdown. Click "Save".
  • You should be able to see the URL of your repository in the "Github Pages" section.
  • Add an index.html file in the root of your repository and you should be able to see it when you view the repository URL.
