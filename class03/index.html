    • Media queries review

    @media (min-width: 768px) {
    // styles until the window reaches 768 pixels or higher in width
    }

    @media (max-width: 768px) {
    // styles until the window reaches 768 pixels in width
    }

    @media (-webkit-min-device-pixel-ratio: 2) {
    // pixel ratio (2 is retina 1 is non-retina)
    }

    @media (min-resolution: 192dpi) {
    // pixel density (another way to test for retina)
    }

    @media (min-width: 768px) and (max-width: 2014px) {
    // combination of conditions
    }

    @media (-webkit-min-device-pixel-ratio: 2), (min-resolution: 192dpi) {
    // a comma separator is a logical operator -- uses one or the other
    }

    You can read more on using media queries on MDN.
