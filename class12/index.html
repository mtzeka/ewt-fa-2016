Reveal.js

11.23.2016

Setup

To install the framework + themes git clone the reveal.js repository.

git clone https://github.com/hakimel/reveal.js.git

You should now have a new directory of "reveal.js" which you will reference for the js and css. You will not make ANY changes to this directory.

<html>

        <link rel="stylesheet" href="reveal.js/css/reveal.css">

        </div>
        <script src="reveal.js/js/reveal.js"></script>
        <script>
            Reveal.initialize();
        </script>
    </body>
</html>

Transitions

Transitions between slides are defined by the `data-transition` attribute.

The possible values are:

  • slide, slide-in, slide-out
  • linear, linear-in, linear-out
  • convex, convex-in, convex-out
  • concave, concave-in, concave-out
  • zoom, zoom-in, zoom-out

View fullscreen

  <div class="reveal">
      <div class="slides">
          <section data-transition="slide">Slide 1</section>
          <section data-transition="zoom-in zoom-out">Slide 2</section>
          <section data-transition="convex">
            <section data-transition="concave">Vertical Slide 1</section>
            <section data-transition="concave">Vertical Slide 2</section>
          </section>
      </div>
  </div>

Transition speed

Transitions speed can be set to "default", "fast" or "slow".

<section data-transition-speed="fast" data-transition="zoom">Slide Content</section>
