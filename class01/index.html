Browsers

Almost everything we do in this class will work in both Chrome and Firefox. Both of these browsers have great standards support and wonderful dev tools.

Before you launch a real-deal web thing, you'd better test it in the other major browsers as well: Safari and IE (and now Edge I guess too?).

Text Editors

The most critical tool when it comes to building websites is your text editor. You're not going to be very productive without being able to write code.

Some of the most common text editors are:

  • Atom
  • Brackets
  • Sublime Text 3
  • Vim

Using Github for class

You are required to get a github account up and running in case you don't already have one. Create a repository dedicated to this class and all your classwork and homework will be uploaded as a new folder in that repository. You will then link to it from your index class file.

All your work should follow the following directory structure:

/css
-- main.css
-- [name].css
/js
-- script.js
-- [name].js
/index.html
