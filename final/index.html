Final Project

Due 12.14.2016

Your turn to teach the class something new!

You will pick a technical topic in web development to research and give the class an 8-10 minute presentation on the topic. Your presentation will include demos, code snippets, and references to all your sources.

You CANNOT talk about something we've already talked about in class. Unless you plan to cover more advanced material on a topic that's already been discussed.

Ideas:

  • SCSS v.s. LESS
  • Lodash v.s. underscore
  • React v.s. Angular (or backbone, ember, vue)
  • React Native
  • React Router
  • Gulp v.s. Grunt v.s. Webpack
  • Package managers - NPM, Bower, Homebrew
  • Compare different text editors and their capabilities
  • Compare sorting algorithms and their performance
  • Advanced Github workflow
  • Advanced SCSS usage
  • Socket.io
  • WebGL
  • Service workers
  • Data scraping
  • A framework we haven't talked about
  • Tell us about an API you tried using
  • A guide to using an open source JS or CSS library
  • Something in MDN we haven't talked about yet

This is just a list to get you started. Choose any topic on the list or anything you come up with on your own.

View this article as an example of the detail & presentation required.

Presentation (20 pts)

Your presentation should be between 8-10 minutes (you will be graded on duration).

Use reveal-js or slides for your slideshow (if there will be one). Powerpoint will not be accepted. But you are free to use google slides.

Instructions/Documentation (30 pts)

Whatever it is that you're teaching us, make sure to provide instructions to a brand new user who knows nothing about your topic. What should be installed in advance? What is the browser compatibility? etc

Demo (50 pts)

Whether is live coding or a demo you've put together in advance, we want to see what you're showing us in action.

Content (50 pts)

Make sure to research your topic thoroughly and make sure to provide 3-5 cited resources (at the end of your slides).

Total: 150 pts
