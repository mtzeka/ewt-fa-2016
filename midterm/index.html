Mobile, Tablet, and Desktop.

Variables, mixins, nesting.

Cross browser compatibility (30 pts)

Must look the same in Chrome, Firefox, IE/Edge, and Safari.

Markup (20 pts)

Proper use of HTML elements.

Needs to look clean and tidy. No use of default fonts or default link colors.

Content (10 pts)

List of all classwork & homework. We want to include two links -- one to the source code on github and one to the URL on our site.

When you turn in your assignment you will receive feedback from me via email. After receiving feedback you have one week to implement the feedback. If you receive a second round of feedback - you have an additional week to complete the second round of feedback.
